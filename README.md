## What is Jobim?

Jobim is an orchestration system that coordinates multiple specialized AI agents to develop complete projects. Instead of chatting with a single assistant, Jobim delegates tasks to specialized agents and synthesizes the results.

## Agents

| Agent | Model | Specialty |
|-------|-------|-----------|
| **Jobim** | Opus | Orchestration, strategy, synthesis |
| **Scout** | Haiku | Research, market analysis, discovery |
| **Builder** | Sonnet | Code, architecture, implementation |
| **Tester** | Sonnet | Tests, code review, security |
| **Shipper** | Sonnet | CI/CD, Docker, deploy |
| **Launcher** | Sonnet | Landing page, README, marketing |
| **Designer** | Sonnet | UI Design, visuals, colors, typography |
| **UXer** | Sonnet | UX Design, flows, usability, behavior |
